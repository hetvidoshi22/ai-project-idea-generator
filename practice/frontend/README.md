# Frontend Practice - Calculator with History

## What I Built
A simple calculator that:
- Performs addition, subtraction, multiplication, and division
- Takes two numbers as input
- Displays the result
- Shows the last 5 calculations
- Handles divide-by-zero errors

## Features
- Input validation
- Error handling
- Dynamic history update
- Clean UI with CSS styling

## What I Learned
- DOM manipulation in JavaScript
- Handling user input
- Conditional logic using switch
- Managing state using arrays
- Limiting history to last 5 entries

## How to Run
1. Open index.html in your browser
2. Enter two numbers
3. Select an operation
4. Click Calculate
